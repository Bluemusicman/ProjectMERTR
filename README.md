# Kurulum
[`MapEditorReborn.dll`](https://github.com/Michal78900/ProjectMER/releases/latest) dosyasını `EXILED/Plugins` klasörüne yerleştirin.
Eklenti yüklendiğinde `EXILED/Configs/ProjectMER` dizinini oluşturacaktır. Bu dizin iki adet alt klasör içerecektir: **Schematics** ve **Maps**
