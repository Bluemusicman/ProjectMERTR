# Kurulum
[`MapEditorReborn.dll`](https://github.com/Michal78900/ProjectMER/releases/latest) dosyasını `LabAPI/plugins` klasörüne yerleştirin.
Eklenti yüklendiğinde `LabAPI-beta/configs/ProjectMER` dizinini oluşturacaktır. Bu dizin iki adet alt klasör içerecektir: **Schematics** ve **Maps**
